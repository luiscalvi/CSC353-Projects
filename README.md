Assignments:

emotion-counter: A simple React app built with Vite. The program has 5 different emotions shown with a simple counter button. 
Users can click the button and the app updates the counter and background. Assignment focused on state mangement, JSX, and component strucutre in React.

todo-app: A simple JavaScript to-do app that allows the user to add a task, complete it, and delete it after. It uses a JSON server as 
the backend database to store the tasks. The app sends API requests to the server to create, update, or remove a task while updating the page dynamically.
This project focuses on working with DOM manipulation, REST requests, and async JavaScript functions.
